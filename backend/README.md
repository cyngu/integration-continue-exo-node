## Description

Backend in NestJS with mongoDB, it needs to start db from the docker-compose.yml.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Swagger

To read swagger documentation, start backend and go to http://localhost:3000/api-docs.
